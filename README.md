Open-Ascii is a simple graphics engine written in pure c which outputs in ascii to a terminal

Features:
- Draw Background
- Draw Pixel
- Draw Line
- Draw Rectangle
- Draw Triangle
- Draw Polygon
- Pixel cordinates to proportional cordinates
