OpenAscii is a simple graphics engine written in pure c which outputs ascii to a terminal

Features:
- Pixel cordinates to proportional cordinates
- Perspective
- Draw Background
- Draw Pixel
- Draw Line
- Draw Triangle
- Draw Tetrahedron
- Draw Rectangle
- Draw Cuboid
